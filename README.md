# Frota Control

Sistema de gestão de frota de caminhões — abastecimentos, fretes, multas, comissões e fechamentos mensais.

## Tecnologias
- HTML/CSS/JavaScript puro (PWA)
- Supabase (PostgreSQL + Auth + RLS)

## Como usar
1. Acesse o sistema pelo navegador
2. Faça login com sua conta
3. Cadastre caminhões, motoristas e comece a registrar operações
